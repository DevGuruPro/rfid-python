# RFID-PYTHON
Software for reading live tag and gps module.

# Getting Started

- Clone this repository:
    ```shell
    cd ~
    git clone https://https://github.com/prosperinfotech/rfid-python
    ```
- Install python libraries

    ```shell
    cd CCKC_EOL
    pip install -r requirement.txt
    ```

- And start!

    ```shell
    python main.py
    ```
